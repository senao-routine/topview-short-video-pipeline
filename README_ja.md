# Topview AI スキル

自然言語で指示するだけで、プロ品質の動画・画像・トーキングアバター・音声を生成できるAIエージェントスキルです。手動操作は一切不要、学習コストもゼロ。

[Agent Skills 仕様](https://agentskills.io/specification)に準拠しており、Claude Code、Cursor など対応するAIコーディングアシスタントで利用できます。

## 何ができる?

スキルをインストールしたら、AIに作りたいものを伝えるだけ。以下はほんの一例です。

### ひと言で作れる

| あなたが言うこと | 得られるもの |
|-----------------|-------------|
| 「商品写真を5パターン、違うスタイルで作って」 | 5枚のAI生成画像（並列バッチ処理） |
| 「夕暮れのビーチを歩く猫の10秒動画を作って」 | AI生成の動画クリップ |
| 「この写真を使って、新商品を紹介する動画を作って」 | リップシンク付きトーキングアバター動画 |
| 「このペットボトルをモデルに持たせて」 | モデルが商品を持つ合成画像 |
| 「この文章を温かみのある女性の声で読んで」 | テキスト読み上げ音声ファイル |
| 「この音声サンプルから私の声をクローンして」 | 以降のナレーションに使えるカスタムボイス |
| 「この商品写真の背景を消して」 | 背景なしのきれいな切り抜きPNG |
| 「この写真の背景をトロピカルビーチに変えて」 | AI編集された画像 |
| 「この参考動画のスタイルで新しい動画を作って」 | スタイル転用された動画 |

### 組み合わせワークフロー

本当の強みは、これらの機能を自由に連結できること。ゴールを伝えれば、AIが全工程を自動でオーケストレーションします。

**ECサイト商品マーケティングキット** — 「商品写真があるので、マーケティング素材一式を作って。」AIが背景除去 → 複数ポーズのモデル展示画像生成 → 商品紹介アバター動画 → 各種アスペクト比のSNSバナーをまとめて作成。1回の会話で完全なマーケティング素材ライブラリが完成。

**教育動画シリーズ** — 「気候変動についてのこの記事を動画シリーズにして。」AIがセグメントごとの台本作成 → キーコンセプトのイラスト生成 → アバタープレゼンター動画 → 選んだ声でナレーション。1記事から複数動画。

**多言語コンテンツ** — 「この商品デモの台本を英語・日本語・スペイン語版で作って。」AIが各言語で音声を生成し、同じ肖像写真で言語別のアバター動画を作成。1つの台本から即座に多言語コンテンツ。

**絵コンテから動画** — 「この4シーンで30秒のアニメーション広告を作りたい。」AIがシーンごとに画像を生成 → 各画像を動画クリップに → 編集用の連続クリップとして納品。テキスト記述からモーションへ。

**ブランド統一コンテンツ一括生成** — 「今週のSNS投稿10本を、うちのブランドスタイルで作って。」AIがビジュアルスタイルの基準画像を作成し、同じスタイルで各トピックのバリエーションを一括生成。コンテンツカレンダー全体でブランド統一。

**デジタルスポークスパーソン** — 「毎回撮影せずに自分のチャンネルの顔になりたい。」肖像写真1枚と声のクローンをアップロードし、各エピソードの台本を渡すだけ。AIがあなたの姿と声でアバター動画を生成し、AI生成イラストをBロールに。

**ナレーション付き製品デモ** — 「このスクリーンショットでアプリ機能を紹介して。」AIが機能説明からナレーションを生成 → スクリーンショットを動画クリップ化 → アバターによるイントロ/アウトロを作成。静止画からプロ品質のデモへ。

### 使い方

1. **インストール** — `npx skills add topviewai/skill`
2. **ログイン** — AIに「ログインして」と伝えるだけ（初回のみ）
3. **作成** — 作りたいものを自然言語で伝える。AIが最適なツール・モデル・パラメータを自動選択。

**あなたの想像力だけが限界** — 上記の例はほんの出発点です。画像生成、動画作成、アバター、ボイスクローン、背景除去などを自由に組み合わせて、思いつくままのワークフローを構築できます。あなたのクリエイティブなビジョンを言葉にするだけで、AIが実現方法を考えます。

## 利用可能なモジュール

内部では以下のモジュールが動いています。直接呼び出す必要はありません — 作りたいものを伝えれば、AIが適切なツールを選びます。

| モジュール | 説明 |
|-----------|------|
| **動画生成（Video Gen）** | 画像→動画、テキスト→動画、参照素材ベースの動画生成。対応モデル: Seedance 2.0 / Sora 2 / Kling 3 / Veo 3.1 / Vidu Q3 / wan2.7 |
| **AI画像（AI Image）** | テキスト→画像、AI画像編集（10以上のモデル対応、最大4K）。対応モデル: Nano Banana 2 / Seedream 5.0 / GPT Image 1.5 / Imagen 4 / Kontext-Pro / Grok Image |
| **口播アバター（Avatar4）** | 写真1枚+台本でリアルなトーキングヘッド動画を生成。多言語・字幕対応、最大120秒/セグメント |
| **背景除去（Remove BG）** | 画像の背景をワンクリック除去 |
| **商品モデル画像（Product Avatar）** | 商品画像をモデルテンプレートに合成し、EC向け展示画像を生成 |
| **テキスト読み上げ（Text2Voice）** | テキストから自然な音声を生成（多言語対応） |
| **ボイス（Voice）** | ボイス一覧・検索、ボイスクローン、カスタムボイス削除 |
| **ボード（Board）** | 生成した作品をWeb上で整理・閲覧・編集・ダウンロード |
| **ユーザー（User）** | クレジット残高・利用履歴の確認 |

## 生成時間の目安

| タスク | モデル | 所要時間 |
|-------|-------|---------|
| 動画 | Standard / Fast (Seedance 2.0) | 約5〜10分 |
| 動画 | その他のモデル (Kling, Sora, Veo, Vidu等) | 約3〜5分 |
| 画像 | GPT Image 1.5 | 約1分 |
| 画像 | その他のモデル (Nano Banana, Seedream, Imagen等) | 約30秒〜1分 |
| アバター | Avatar4 | 約2〜5分（台本の長さによる） |
| 音声 | Text2Voice | 約10〜30秒 |
| 背景除去 | Remove BG | 約10〜30秒 |
| 商品モデル | Product Avatar | 約1〜2分 |

## セットアップ

Python 3.8以上と Topview AI アカウントが必要です。

```bash
npx skills add topviewai/skill
pip install -r scripts/requirements.txt
```

ログインはAIアシスタントに「ログインして」と伝えるだけ。ログイン用リンクがチャットに表示されるので、ブラウザで開いてサインインすれば完了です（初回のみ）。
